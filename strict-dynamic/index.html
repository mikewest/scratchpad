strict-dynamic at work:

This page has a CSP of script-src 'nonce-abcdefg' 'strict-dynamic'. <script src="./script.js" nonce="abcdefg"></script> will load, and that it will be able to load additional scripts (which, in turn, can load additional scripts) so long as those scripts are loaded via non-"parser-inserted" <script> elements.

Note that this mechanism requires the use of either nonces or hashes to restrict the initial scripts that are directly loaded. If you need to constrain the sources from which scripts are loaded as well, you can do so via an additional policy which has an allowlist of hosts (e.g. Content-Security-Policy: script-src 'nonce-abcdefg' 'strict-dynamic', script-src https://my.amazing.cdn).
